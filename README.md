# Real-Time Face Expression Detection

A real-time face expression detection system built using Python, OpenCV, and DeepFace.

## Features

- Live webcam face detection
- Real-time expression analysis
- Displays detected expression on screen
- Easy to run and customize

## Technologies Used

- Python
- OpenCV
- DeepFace
- TensorFlow
